# Smart Fashion Styling Recommendation System

A Python-based console application that provides outfit recommendations based on:
- Gender
- Occasion
- Favorite color
- Weather

## Features
- Personalized outfit suggestions
- Color matching recommendations
- Fashion tips
- Hairstyle and accessory suggestions

## Technologies Used
- Python
- Conditional Statements
- Lists
- Dictionaries
- Random Module
